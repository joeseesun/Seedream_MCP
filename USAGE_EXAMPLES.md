# Seedream MCP 使用示例

## 🎨 在 MCP 客户端中使用

### 简单提示词

直接说你想要什么:

```
"生成个小猫"
"画一只小狗"
"来一张风景图"
```

### 详细提示词

提供更多细节获得更好的效果:

```

"日落时分的海边,波光粼粼,温暖的色调,电影感"
```

## 🔧 高级用法

### 指定图片尺寸

```
"生成一张 4K 分辨率的风景图"
```

支持的尺寸:
- `2K`: 2304x1728 (默认)
- `4K`: 4608x3456
- `1:1`: 2304x2304
- `16:9`: 2304x1296
- `9:16`: 1296x2304

### 去除水印

```
"生成个小猫,不要水印"
```

## 💡 使用技巧

### 1. 提示词建议

**好的提示词:**
- ✅ "一只可爱的橘色小猫咪,坐在窗台上,阳光洒在身上,卡通风格"
- ✅ "日落时分的海边,波光粼粼,温暖的色调,电影感"
- ✅ "现代简约风格的客厅,落地窗,绿植,自然光"

**避免:**
- ❌ 过于简单: "猫"
- ❌ 过于复杂: 超过 600 字的长篇描述
- ❌ 矛盾的要求: "写实风格的卡通人物"

### 2. 风格关键词

- **卡通风格**: 可爱、色彩鲜艳
- **写实风格**: 真实、细节丰富
- **电影感**: 戏剧性光影、构图
- **水彩风格**: 柔和、艺术感
- **油画风格**: 厚重、质感

### 3. 光线描述

- **自然光**: 柔和、真实
- **阳光**: 温暖、明亮
- **黄昏**: 温暖、浪漫
- **夜晚**: 神秘、宁静
- **逆光**: 轮廓光、氛围感

## 🆘 常见问题

### Q: 为什么图片没有显示?

A: 检查以下几点:
1. 客户端是否支持 Markdown 渲染
2. 七牛云 URL 是否可以访问
3. 网络连接是否正常

### Q: 可以生成多张图片吗?

A: 目前每次生成一张图片。如需多张,可以多次调用。

### Q: 图片会过期吗?

A: 
- 本地保存: 永久保存
- 七牛云: 永久保存(除非手动删除)

### Q: 如何分享图片?

A: 直接分享七牛云 URL 即可,例如:
```
https://newimg.t5t6.com/seedream/20251118_102119_生成个小狗_20251118_102119.jpeg
```
